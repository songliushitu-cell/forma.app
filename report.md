# FORMA 骨盆评估 - Pro — 详细产品报告

## 1. 产品概述
FORMA 骨盆评估 - Pro 是基于浏览器端姿态检测模型的前端体态分析工具，旨在为普通用户提供便捷的骨盆与下肢角度测量和初步判读，帮助用户进行早期自检与训练参考。强调：本工具不提供诊断或医疗建议。

## 2. 核心功能列表
- 预设动作：SLR（直腿抬高）、髋屈、肩屈 等；
- 输入方式：实时摄像头、相册上传、手动录入数据；
- 视觉交互：Canvas 覆盖实时关键点、角度与标注；
- 微调工具：锁定帧、手动移动关键点以修正检测；
- 报表生成：数值（角度、左右差）、匹配度、判读文字、可保存截图。

## 3. 技术架构概览
- 前端：单页静态应用（index.html），UI 使用 Tailwind CSS；
- 模型：使用 TensorFlow.js 与 pose-detection（通过 CDN 加载），支持 WebGL 与 CPU 后端；
- 数据流：摄像头/图片 → Pose 模型推理 → 关键点计算（角度/差值）→ 可视化与报告生成；
- 部署：静态页面托管即可（需保证外网访问 CDN）。

## 4. 运行与安装要求
- 设备：具备摄像头的电脑或手机；
- 浏览器：Chrome/Edge/Safari 最新版本（建议使用 Chrome）；
- 权限：需允许摄像头权限；
- 网络：需联网以加载外部模型与库（暂不支持完全离线）。

## 5. 精度限制与已知风险
- 模型精度受光照、服装遮挡、摄像头角度与分辨率影响；
- 不适用于替代专业临床评估；
