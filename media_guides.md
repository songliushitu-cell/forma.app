# 截图与短视频脚本建议

## 一、截图建议（用于社媒与文章）

1. 封面：实时检测界面全屏
   - 文件名建议：cover_realtime.png
   - 说明：包含 video/canvas 实时关键点、角度标签与评分卡（若有）。裁剪建议：1080×1080（正方形）或 1200×628（横向）。加标题："FORMA 骨盆评估 - Pro"（下方留出 120px 文本区）。

2. 步骤示意图：手机侧放与拍摄位置
   - 文件名建议：step_position.png
   - 说明：展示手机/三脚架放置位置、与被测者的相对距离与角度（标注“0.5-1.5m”，“镜头高度与髋部持平”）。裁剪建议：800×1200（竖图）。

3. 实时检测细节截屏
   - 文件名建议：detection_detail.png
   - 说明：截取关键点与角度标注放大视图（显示锁定帧/微调控件）。用于说明“微调”功能。裁剪建议：800×800 或 1024×512。

4. 报告页/评分卡截屏
   - File: report_card.png
   - 说明：截取最终报告区域，突出匹配度、左右差值与判读摘要。建议裁剪为 900×600（横向），并模糊或占位联系方式以保护隐私。

5. 导出/保存示例
   - 文件名建议：export_example.png
   - 说明：显示“保存为图片/PDF”后的保存对话或生成文件预览，便于用户理解保存流程。

**截图拍摄与美化建议**
- 统一色调与字体；在封面添加 1-2 行简短口号（如“60 秒快速自检骨盆体态”）。
- 保留空白区域放置按钮/CTA 文案（便于不同社媒适配）。


## 二、短视频脚本（15-30 秒，适合抖音/小红书/B站短视频封面）

目标：演示“拍摄 → 微调 → 生成报告”三步，突出快速与易用性。

时长分配建议：总时长 20-25 秒。

镜头分镜（Shot List）

1. 开场 2s — 封面/片头
   - 画面：APP/网页 Logo 或 封面截屏。
   - 文案/字幕："FORMA 骨盆评估 - Pro"（伴随短促音乐）。

2. 准备示意 4s
   - 画面：手机侧放示意（step_position.png），箭头标注“侧面全身入镜”。
   - 旁白/字幕："把手机放在侧面，穿紧身服装"。

3. 实时捕捉 6-8s
   - 画面：屏幕录制显示实时检测关键点、角度数值随动作变化（加速播放 1.5x）。
   - 旁白/字幕："启动实时捕捉，按提示完成动作"。

4. 微调与锁定 3-4s
   - 画面：演示手动微调关键点并锁定帧，角度显示更新。
   - 旁白/字幕："若关键点偏移，可手动微调提高精度"。

5. 生成报告 3-4s
   - 画面：生成报告页面切换，展示评分卡与简短判读摘要。
   - 旁白/字幕："自动生成可视化报告，快速查看左右差值"。

6. 结尾 CTA 2-3s
   - 画面：展示“体验链接/扫码”与平台占位（小红书/抖音/B站）。
   - 文案/字幕："立即体验 →（链接/扫码）"，并加上免责声明小字“仅供参考，非医疗诊断”。

配乐与音效
- 背景音乐节奏明快、清新；音量在旁白之下。关键操作时添加轻快点击音效（如拍照、完成报告）。

旁白样本文案（可选）
- "FORMA 骨盆评估，60 秒快速自检。"
- "把手机放侧面，按提示完成动作。"
- "可微调关键点，生成匹配度报告。"
- "结果仅供参考，若有不适请就医。"

字幕与标签
- 视频底部常显字幕（简短指令），结尾显示话题标签：#体态自检 #骨盆评估 #家庭康复

导出规格建议
- 分辨率：1080×1920（竖屏优先），亦可 1080×1080 用于其他平台；
- 时长：建议 15-30 秒；
- 字幕：16-20 pt，保证移动端可读。
